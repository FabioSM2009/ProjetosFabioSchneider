Tabelas:

Primeiramente, devemos saber do que é composto uma tabela.




Uma tabela HTML pode ser definida com a tag "table", cada linha da tabela é definida com a tag "tr", cada célula é definida com a tag "td", cada cabeçário é definido com a tag "th".

É possível utilizar palavras-chaves também, como: width, height, border, colspan, rowspan, etc.
